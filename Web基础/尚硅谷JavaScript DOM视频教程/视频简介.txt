视频简介：

   JavaScript DOM 编程是编写 Ajax 程序的必备技术，也是 Java 程序员必须掌握且熟练运用的技术之一。本视频教程涵盖 JavaScript DOM 编程的全部核心技术。

视频特点：

1. 延续了佟刚老师深入浅出、代码驱动的授课风格
2. 2 天的授课中涉及到 10 余个真实的企业应用案例
3. 自定义 DOM 标准中没有的节点互换 replaceEach(aNode，bNode)、在节点后插入新节点 insertAfter(newNode, refNode) 方法
4. 部分案例使用了 XPath 技术
